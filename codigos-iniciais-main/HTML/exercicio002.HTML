exportar imagens para o site

para usar uma imagem primeiro coloque ela na mesma pasta que o codigo e apos utilizar o comando < img src = "" alt= "" > após isso entre as aspas de src= "" aperte CTRL e ESPAÇO e clique no nome da imagem desejada

o src do comando significa souce e o alt serve apenas para descrição da imagem.
tambem pode-se colocar imagens que estao em sub-pastas, ultizando o mesmo comando CTRL e ESPAÇO e clicando na pasta em que esta localizada a imagem
tambem podemos carregar imagens externas, copiando o endereço delas qua apareçe ao clicar com o botão direito.
ja dentro do codigo utiliza-se o CTRL + V entre as aspas de src para endereçar a imagem.

estas sao as tecnicas mais ultilizadas.
